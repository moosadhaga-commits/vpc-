#  AWS VPC Project

## Objective
Create and configure a Virtual Private Cloud (VPC) in AWS.

## Services Used
- Amazon VPC
- Public Subnet
- Private Subnet
- Internet Gateway
- Route Table

## Steps Performed
1. Created a VPC.
2. Created public and private subnets.
3. Attached an Internet Gateway.
4. Created a Route Table.
5. Associated the Route Table with the public subnet.

## Outcome
Successfully created a VPC with public and private subnets.

## Skills Learned
- VPC
- Subnets
- Internet Gateway
- Route Tables
- Basic AWS Networking
